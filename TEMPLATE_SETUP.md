# Инструкция по настройке шаблона репозитория

## Как сделать репозиторий шаблоном на GitHub

1. Откройте ваш репозиторий на GitHub
2. Перейдите в **Settings** (Настройки)
3. Прокрутите вниз до секции **Template repository**
4. Установите галочку **Template repository**
5. Сохраните изменения

Теперь на странице репозитория появится кнопка **"Use this template"**, которая позволит другим пользователям создавать новые репозитории на основе вашего шаблона.

## Что включено в шаблон

- ✅ Базовая структура Python-функции для Yandex Cloud
- ✅ GitHub Actions workflow для автоматического деплоя
- ✅ Скрипты для локального тестирования
- ✅ Документация по настройке
- ✅ Правила Cursor IDE (если используются)

## Рекомендации

После включения шаблона:

1. Обновите `README.md` с описанием шаблона
2. Убедитесь, что все файлы, которые не должны попадать в новый репозиторий, находятся в `.gitignore`
3. Рассмотрите добавление `.github/template.yml` для переменных шаблона (уже создан)
